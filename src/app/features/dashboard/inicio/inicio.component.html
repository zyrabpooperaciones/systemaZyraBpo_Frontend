Guía de Flujo Operativo

Entiende los pasos de configuración del procesamiento automático de archivos Excel.

1

Definir el Catálogo General

Disponible

Asigna las equivalencias de cabeceras de Excel de tus catálogos de Datos, Montos y prioridades de Teléfonos. Solo las columnas activadas aquí podrán ser utilizadas para armar las plantillas.

2

Crear Plantillas de Importación

Disponible

Agrupa columnas activas bajo nombres personalizados (ej. subida de saldos, deudas mensuales) asociando la relación Many-to-Many que requiera el archivo específico de esa cobranza.

3

Cargar Archivo Excel y Procesar

Próximamente

Importa los registros de deudas del archivo Excel utilizando la plantilla configurada. El trigger de la base de datos recalculará y actualizará automáticamente los saldos de cobranza.
